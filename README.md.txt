UPDATE 10/05/2017

Pour lancer l'application, téléchargez Android Studio et importez le projet dans l'environnement de travail.

Branchez votre terminal android à l'aide d'un câble USB à votre ordinateur, puis exécutez :

- cliquez sur Run dans la barre des outils
- cliquez sur Run "app"
- séléctionnez votre téléphone dans la liste "Connected Devices"
- cliquez sur "OK"

Ouvrez l'application "Trouver soirée" dans la liste de vos applications si elle n'est pas déjà ouverte.
